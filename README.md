# Scripts Overview

###  Script 1 — System Information

Displays key system details:

* Kernel version
* Current user
* System uptime



###  Script 2 — Git Package Checker

Checks whether **Git** is installed on your system and:

* Displays installation status
* Shows Git version
* Provides basic configuration details



###  Script 3 — Directory Auditor

Audits system directories to show:

* File permissions
* Directory sizes



###  Script 4 — Log Analyzer

Analyzes log files by:

* Searching a keyword
* Counting occurrences



###  Script 5 — Open Source Manifesto Generator

Generates a personalized manifesto based on user input.
